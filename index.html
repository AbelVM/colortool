Color scales builder

This minimal tool helps you build a rightful color scale to be used in data visualizations. Just choose the type of scale, the number of steps, the target background and play with the starting and ending colors, everything else is already managed. You can choose between three perceptual color spaces to perform the internal calculations:
  • Spectral (not exactly a color space)
  • CIE L*a*b*
  • Oklab
  • HCL
Check the tooltips for relevant information when something turns red...
